# University Library (Java)

Console-based Java program that simulates a simple **university library management system**.

## Features
- Add library items:
  - Book (author, ISBN)
  - Magazine (issue number)
  - Digital book (file size, file type)
- Add members:
  - Student member (AM, department)
  - Professor member (specialty, department)
- Search an item by code and print its details
- Borrow / return items
- Print active loans
- Basic library statistics (total items, total members, active loans)

## How to run
1. Compile:

```bash
javac "Java Files"/*.java
```

2. Run:

```bash
java -cp "Java Files" Main
```
